Game Rules

  • A random number is displayed at the start of the game
  • Click on the crystals below to add points to your score. Watch out! The value of each crystal will only be displayed after you click on it!
  • Win the game by matching the random number displayed
  • Go over the random number and you lose!
  • The crystals values reset when a new game starts.
